# 1) WBS Deploy Setup Tool Eval 1 {#adr_0001}

Date: 2026-05-04

## Status

accepted

## Context

The first version of the WBS Deploy Setup Tool UI was evaluated by Keren through comments in a Figma document:

- [WBS Installation Wizard v1 evaluation notes](https://www.figma.com/design/GUWSDKbomRnGZiI5ZIusMl/-WBS--Installation-Wizard-v1-evaluation?node-id=0-1&t=Y8eH9VPeSd34IVSm-1)

The review focused on the usability of the setup flow, including form labels, password visibility, DNS/domain guidance, setup progress, final configuration handling, and areas where the UI needed clearer copy or interaction patterns.

## Decision

Adjust the WBS Deploy Setup Tool UI according to the UX feedback from Eval 1.

The response to the Figma feedback is captured in this Google Sheets document:

- [WBS Deploy Setup Tool UI Notes](https://docs.google.com/spreadsheets/d/1WhroI0WnuB2AHr848sezBzHQNckSHlVH_VYZlwHv11w/edit?usp=sharing)

That document records the individual comments, the response to each comment, and the status of each item.

## Consequences

The setup-tool UI is more usable and clearer for a first-time setup flow. It is ready for another round of testing and review.
